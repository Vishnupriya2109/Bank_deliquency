An intelligent machine learning web application that predicts customer delinquency risk using financial and behavioral data. The system analyzes customer-related inputs and classifies users into different risk levels such as Low Risk, Medium Risk, and High Risk.

Features: 
Machine Learning based delinquency prediction , Real-time risk analysis, Flask backend integration, Interactive frontend UI, Customer risk categorization,Database connectivity for storing predictions, Export prediction results to Excel, End-to-end ML workflow implementation
